Como usar o Football Field Sim

Jogadores
  • Use os painéis Ataque e Defesa para adicionar jogadores com o botão +.
  • Cada jogador recebe uma sigla (até 3 letras). Duplo clique sobre o jogador no campo para editar a sigla.
  • Clique e arraste para mover um jogador pelo campo.
  • Segure Shift ou Ctrl/Cmd e clique para selecionar múltiplos jogadores e movê-los juntos.
Gravação de Rotas
  • Clique direito sobre um jogador para abrir o modal de gravação de movimento.
  • Clique em Iniciar gravação e arraste o jogador para traçar a rota.
  • Clique em Parar gravação e depois Salvar movimento para confirmar.
  • Use o botão 〰 na barra lateral para mostrar ou ocultar todas as rotas.
Animação
  • ▶ — Inicia a animação de todos os jogadores com rotas gravadas.
  • ↺ — Restaura todos os jogadores às posições originais.
  • ✕ — Apaga todos os movimentos gravados sem mover os jogadores.
Formações Rápidas
  • Os botões na barra lateral posicionam automaticamente jogadores em formações pré-definidas.
  • Ataque: 🥪 Linha Ofensiva · IF Formation · ⬅⬅ Trips Stack Esq · ➡➡ Trips Stack Dir · 2x2 Ace · 3x1 Trey TE
  • Defesa: 🐶 Front 4-2 · 👻 Front 3-3 · 🦈 Front 3-4
Salvar e Carregar
  • 💾 — Exporta a formação atual como arquivo .json.
  • 📂 — Carrega uma formação a partir de um arquivo .json salvo anteriormente.
  • 📋 — Abre a galeria de playbooks pré-carregados.
  • 🗑️ — Limpa completamente o campo, removendo todos os jogadores.
